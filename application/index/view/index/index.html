       广东顺德西安交通大学研究院由西安交通大学、广东省科技厅、顺德区人民政府三方本着“优势互补、互利互惠、长期合作、共同发展”的原则共同组建，并于2011年3月7日在广东顺德正式登记注册为事业法人。研究院旨在依托西安交通大学在新能源、先进制造、高端电子信息、新材料等学科领域具有的人才、技术和科研成果等方面有时，集聚各方面资源，结合顺德产业分布特点，围绕当地经济、产业、社会的发展需求，重点在高端装备制造、节能环保、高端电子信息、新材料等领域开展技术创新、人才培养、成果转化服务。通过科技研发与产品中试、技术成果转移转化、存量人才知识结构更新与能力提升，建成国内一流的集科技创新服务、人才培养及成果转化孵化服务等功能于一体的新型研发机构。
